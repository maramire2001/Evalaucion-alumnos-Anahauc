## 🎯 PRÓXIMO PASO: Crear Repositorio en GitHub

### Opción Rápida (5 minutos):

1. **Ve a https://github.com/new**

2. **Rellena así:**
   - Repository name: `evaluacion-maestria-anahuac`
   - Description: "Full-stack Django app with Groq AI for student blogs evaluation"
   - Public (para que Render acceda)
   - NO inicializar con README/gitignore/license

3. **Click "Create repository"**

4. **Copia el comando que aparece** (algo como):
   ```bash
   git remote add origin https://github.com/TU_USUARIO/evaluacion-maestria-anahuac.git
   git branch -M main
   ```
